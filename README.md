# 🏏 IPL Data Analysis Dashboard

## 📊 Project Overview

This project focuses on analyzing IPL (Indian Premier League) cricket data using a complete data analytics workflow.
The goal is to extract meaningful insights about players and matches through interactive dashboards.

---

## 🛠️ Tools & Technologies Used

* Microsoft Excel (Data Cleaning using Power Query Editor)
* SQL (Data Storage & Querying)
* Power BI (Data Visualization & Dashboard Creation)

## 🔄 Data Processing Steps

1. Imported raw IPL dataset into **Excel**
2. Cleaned and transformed data using **Power Query Editor**
3. Loaded cleaned data into **SQL database**
4. Queried data from SQL and connected it with **Power BI**
5. Built interactive dashboards for analysis

## 📈 Dashboards Created

### 🔹 Player Analysis Dashboard

* Total Runs: 305K+
* Total Wickets: 12K+
* Average Strike Rate: 102
* Top 10 Players by Runs
* Top 10 Players by Wickets
* Strike Rate Analysis
* Runs vs Strike Rate comparison

### 🔹 Match Analysis Dashboard

* Total Matches: 1095
* Total Seasons: 16
* Total Teams: 20
* Top Teams by Wins
* Matches per Season
* Matches by City distribution

## 🎯 Key Insights

* Identified top-performing players based on runs and wickets
* Compared strike rate vs performance
* Analyzed team performance across seasons
* Observed match distribution across different cities
